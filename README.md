# NaixiMCServerPing
## 简介
奶昔Minecraft服务器查询器

这是一个简单的界面式Minecraft服务器查询器

总共只有27行代码

源文件大小2KB

## 更多

调用了[FansMCAPI](https://fansmc.com/api/)

使用[Python](https://python.org)编写

使用到以下python库
1. requests
2. os
3. time
4. base64
5. easygui
